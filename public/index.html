Azure static Web Apps

Azure Static Web Apps est un service d'automatisation qui compile et déploie une application web d'un dépot (repository) de code (GitHub ou Azure DevOps) vers Azure. Le flux est attaché à une branche du dépot et déclenché à chaque changement (push commits ou accept pull requests) sur cette branche.

Fonctionnalités

  • Hébergement web pour du contenu statique tel que HTML, CSS, JavaScript et images.
  • Intégration d'API de support via Azure Functions
  • Intégration avec GitHub et Azure DevOps où un changement déclenche la construction et le déploiement.
  • Contenu statique mondialement répandu, mettant le contenu plus proche des utilisateurs.
  • Certificats SSL gratuit, renouvelé automatiquement.
  • Nom de domaine personnalisé peut être configuré.
